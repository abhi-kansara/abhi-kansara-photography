# Abhi Kansara Photography

A high-fidelity, cinematic photography portfolio built with **Next.js 16**. This platform is designed to showcase visual storytelling through premium aesthetics, smooth motion, and high-performance media delivery.

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router & Turbopack)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
- **Icons:** [Lucide React](https://lucide.dev/)

## Key Features

- **Cinematic Morph Carousel:** An atmospheric background system featuring smooth animated blending and scaling between high-resolution shots.
- **High-Fidelity HLS Video:** Advanced video player supporting SmugMug's HLS master streams for adaptive, premium quality playback.
- **Smart Offerings Showcase:** Infinite-scroll services carousel with intelligent user-interaction awareness and seamless resumption.
- **Dynamic Hero Overlay:** Scroll-reactive brightness control that ensures maximum photo impact at the top while maintaining readability as the user scrolls.
- **Premium Custom Scrollbar:** A minimalist, floating scrollbar architecture that provides on-demand visibility without affecting the UI layout.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Viraj-Mavani/abhi-kansara-photography.git
   cd abhi-kansara-photography
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
